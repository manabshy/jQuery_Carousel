Build a re-usable carousel plugin

This task is to build an extensible and reusable horizontally sliding carousel controlled by the 'next' and 'previous' buttons. jQuery is already referenced from this page and the CSS and HTML can be adjusted as required.

The carousel must:

  • be able to support up to 5 items
  • include an indication of the number of items
  • clearly disable the 'next' or 'previous' buttons when relevant
  • be simple to re-use - once built please show 2 independent carousels on one page
  • function correctly in the latest Chrome and Firefox browsers
  • be a lightweight solution - not bloated with unnecessary functional options

The carousel does not have to:

  • continuously loop
  • support flexible widths - a fixed width solution is expected
  • support touch interactions

The aim of this task is to illustrate your ability to write JavaScript code using an object oriented approach.
